# Game Pillars

## One-Sentence Pitch

이 게임을 한 문장으로 설명한다.

## Player Fantasy

플레이어가 어떤 존재가 되고 싶은지, 어떤 감각을 기대하는지 적는다.

## Core Pillars

- Pillar 1: 반드시 지켜야 하는 핵심 재미.
- Pillar 2: 플레이 중 반복해서 드러나야 하는 선택.
- Pillar 3: 다른 기능보다 우선되는 피드백 방향.

## Non-Goals

- 첫 프로토타입에서 검증하지 않을 기능.
- 레퍼런스에는 있지만 지금은 복사하지 않을 기능.

## Success Criteria

- 30초 안에 반복 가능한 재미 검증 기준.
- 플레이어가 성공/실패를 이해하는 기준.
